# Todo/Project Management - Starter Codebase

This is the starter codebase for the Todo/Project Management capsule project.

## Structure

- `frontend/` - React frontend with drag-and-drop support
- `backend/` - Node.js/Express backend
- `database/` - Database schema for tasks, projects, teams

## What's Included

- Basic project structure
- Package.json files with drag-and-drop libraries
- Database schema template
- Basic configuration

## What You Need to Build

- Task CRUD operations
- Project organization
- Kanban board with drag-and-drop
- Team collaboration
- Task assignment
- Filtering and search
- Real-time updates (optional)

## Getting Started

1. Review requirements in main README.md
2. Install dependencies
3. Set up database
4. Start with basic task list
5. Add Kanban board
6. Add collaboration features
