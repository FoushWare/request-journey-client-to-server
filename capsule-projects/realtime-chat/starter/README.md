# Real-time Chat - Starter Codebase

This is the starter codebase for the Real-time Chat capsule project.

## Structure

- `frontend/` - React frontend with Socket.io client
- `backend/` - Node.js/Express backend with Socket.io server
- `database/` - Database schema for messages, rooms, users

## What's Included

- Basic project structure
- Package.json files with Socket.io
- Database schema template
- Basic WebSocket setup

## What You Need to Build

- WebSocket server setup
- Real-time messaging
- Room/channel management
- User presence tracking
- Typing indicators
- Message history
- File sharing (optional)

## Getting Started

1. Review requirements in main README.md
2. Install dependencies
3. Set up WebSocket server
4. Implement basic messaging
5. Add rooms and features
6. Optimize for scale
