# API Gateway - Starter Codebase

This is the starter codebase for the API Gateway capsule project.

## Structure

- `gateway/` - API Gateway service (Express.js or Kong)
- `services/` - Sample microservices (user-service, product-service, etc.)
- `database/` - Database schemas for each service

## What's Included

- Basic gateway structure
- Sample microservice templates
- Database schema templates
- Basic configuration

## What You Need to Build

- Request routing
- Authentication (JWT, API keys)
- Rate limiting
- Service discovery
- Load balancing
- Request/response transformation
- Monitoring and logging

## Getting Started

1. Review requirements in main README.md
2. Set up gateway service
3. Create sample microservices
4. Implement routing
5. Add authentication
6. Add rate limiting
7. Deploy to Kubernetes
