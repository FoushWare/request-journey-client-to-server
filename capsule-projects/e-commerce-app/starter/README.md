# E-commerce App - Starter Codebase

This is the starter codebase for the E-commerce App capsule project.

## Structure

- `frontend/` - React/Next.js frontend (basic structure)
- `backend/` - Node.js/Express backend (basic structure)
- `database/` - Database schema and migrations

## What's Included

- Basic project structure
- Package.json files with dependencies
- Database schema template
- Basic configuration files

## What You Need to Build

- User authentication system
- Product catalog
- Shopping cart
- Checkout process
- Order management
- Admin panel (optional)

## Getting Started

1. Review the project requirements in the main README.md
2. Set up your development environment
3. Install dependencies: `npm install` in frontend and backend
4. Set up database using the schema in `database/`
5. Start building features incrementally
