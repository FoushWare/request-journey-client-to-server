# Starter: Argo CD Setup

This directory contains the starting point for setting up Argo CD.

## What You'll Need

- Kubernetes cluster access
- Git repository with Notes App manifests
- kubectl configured

## Your Task

1. Install Argo CD in your cluster
2. Access Argo CD UI
3. Connect Git repository
4. Create Argo CD Application for Notes App
5. Enable auto-sync

## Next Steps

After completing the task, compare your solution with `../final-solution/`.
