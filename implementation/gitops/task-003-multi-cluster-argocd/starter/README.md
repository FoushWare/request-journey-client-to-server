# Starter: Multi-Cluster Argo CD

This directory contains the starting point for multi-cluster and multi-environment Argo CD setup.

## What You'll Need

- Multiple Kubernetes clusters OR multiple namespaces
- Argo CD installed
- Git repository organized by environment

## Your Task

1. Register additional clusters (if using multi-cluster)
2. Organize Git repository for multiple environments
3. Create Applications per environment
4. Configure environment promotion
5. Set up load balancing (if multi-cluster)

## Next Steps

After completing the task, compare your solution with `../final-solution/`.
