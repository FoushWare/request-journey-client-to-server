# Starter: Microservices Architecture

This directory contains the starting point for refactoring Notes App into microservices.

## Current State

- Monolithic backend handling all functionality
- Single database
- All features in one codebase

## Your Task

1. Define service boundaries (Auth, Notes, Users)
2. Create separate service structures
3. Implement Auth Service
4. Implement Notes Service
5. Implement User Service
6. Set up inter-service communication
7. Configure NGINX as API Gateway
8. Deploy to Kubernetes
