# Starter: Istio Observability

This directory contains the starting point for setting up Istio observability.

## What You'll Need

- Istio installed (task-001)
- Prometheus installed (from logging phase)
- Grafana installed (from logging phase)
- Optional: Jaeger for tracing

## Your Task

1. Configure Prometheus for Istio metrics
2. Create Grafana dashboards
3. Enable distributed tracing with Jaeger
4. Install Kiali (optional) for service graph

## Next Steps

After completing the task, compare your solution with `../final-solution/`.
