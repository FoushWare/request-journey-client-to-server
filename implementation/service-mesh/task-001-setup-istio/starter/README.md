# Starter: Istio Service Mesh Setup

This directory contains the starting point for setting up Istio.

## What You'll Need

- Kubernetes cluster (4GB+ RAM recommended)
- Notes App deployed in Kubernetes
- kubectl configured

## Your Task

1. Install Istio CLI
2. Install Istio in cluster
3. Enable automatic sidecar injection
4. Onboard Notes App services
5. Verify service mesh is working

## Next Steps

After completing the task, compare your solution with `../final-solution/`.
