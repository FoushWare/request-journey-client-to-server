# Starter: Istio Security and mTLS

This directory contains the starting point for securing Notes App with Istio mTLS.

## What You'll Need

- Istio installed (task-001)
- Notes App deployed with Istio sidecars

## Your Task

1. Enable mutual TLS (mTLS) for all services
2. Create PeerAuthentication policy
3. Create AuthorizationPolicies
4. Verify zero-trust security
