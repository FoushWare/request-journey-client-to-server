# Task 013: Integrate SonarQube in Pipelines - Implementation Directory

This directory contains implementation examples for integrating SonarQube into CI/CD pipelines.

## Structure

- `starter/`: Starting point (empty)
- `final-solution/`: Complete solution with:
  - `jenkins/Jenkinsfile.sonarqube`: Jenkins pipeline with SonarQube
  - `github-actions/.github/workflows/sonarqube.yml`: GitHub Actions workflow
  - `README.md`: Integration guide

## Usage

1. Review the task: `tasks/ci-cd/task-013-integrate-sonarqube-in-pipelines.md`
2. Update your Jenkinsfiles or GitHub Actions workflows
3. Configure SonarQube server/credentials
4. Reference `final-solution/` for examples
