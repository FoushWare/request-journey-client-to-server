# Task 003: Create Docker Compose for Notes App

## Structure

This directory contains:
- `starter/` - Notes App with frontend and backend Dockerfiles (from task-002 final-solution)
- `final-solution/` - Complete solution with docker-compose.yml orchestrating all services

## Usage

1. **Read the task**: Review `tasks/docker/task-003-create-docker-compose-for-notes-app.md`
2. **Start with starter/**: Copy `starter/` to your working directory
3. **Implement the task**: Create docker-compose.yml to orchestrate frontend, backend, and database
4. **Compare with final-solution/**: After completing, review `final-solution/` to see the complete implementation

## Starter Contents

The `starter/` directory contains:
- Frontend with Dockerfile
- Backend with Dockerfile
- Database schema
- **No docker-compose.yml** - you need to create it!

## Final Solution Contents

The `final-solution/` directory contains:
- Complete Notes App with docker-compose.yml
- All services orchestrated (frontend, backend, database)
- Network configuration
- Volume configuration
- All necessary configuration files
